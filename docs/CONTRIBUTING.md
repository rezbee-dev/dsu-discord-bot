## Setting up

**With Docker**
- With `make` installed
  - run `make dev`
- Without `make` installed
  - run `docker compose -f ./compose.dev.yaml up --build`

**Without Docker**
- In Intellij
  - Click on `Main` file in `src` directory and run
- In maven
  - Run `./mvnw clean install` then `./mvnw spring-boot:run`
